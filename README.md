# LeadConnect

Production-ready multi-tenant SaaS platform blueprint for LinkedIn lead generation, outreach automation, CRM, analytics, billing, and AI sales assistance.

## Stack
- Backend: Laravel 12, PHP 8.4, MySQL 8, Redis, Horizon, Sanctum, Reverb, Scout, Meilisearch
- Frontend: Vue 3, TypeScript, InertiaJS, Pinia, Tailwind CSS, shadcn-vue
- Infra: Docker, Nginx, Supervisor, AWS S3, Cloudflare, GitHub Actions
- Billing: Stripe, Paddle
- AI: OpenAI, Anthropic
